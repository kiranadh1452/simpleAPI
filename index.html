Simple API

Documentation for the Simple API

This is a simple API that allows you to create a new user and get a list of all users.

Getting all users

GET /api/users
Returns an object containing all users (Object of many objects)

Getting user by id

GET /api/user/id=value
Replace value with desired id Returns an object containing the user with the given id (Object)

Getting user by name

GET /api/user/name=value
Replace value with desired name Returns an object containing the user with the given name (Object)
